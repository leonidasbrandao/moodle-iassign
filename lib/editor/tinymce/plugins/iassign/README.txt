.: About :.

This is the iAssign (Interactive Assignment) button for insert interative activities in TinyMCE editor.


.: Quick install instructions (to be used by the system administrator) :.

0) Be sure you have (at least) Moodle "2.3 or more" installed.
1) Be sure to have the latest language package, such as English (en), Portuguese (pt_br), Spanish (es), or French (fr) (these are the current languages for iAssign).
2) Be sure to have the latest version of the module iAssign (see in http://www.matematica.br/tinymce_iassign).
3) Unpack 'tinymce_iassign.x.x.zip' (or 'tinymce_iassign.x.x.tgz').
4) Copy the 'iassign' directory into the "lib\editor\tinymce\plugins\" subdirectory of Moodle installation.
5) Under role "administrator", go to the 'Site administration' and click on 'Notifications'.
6) If button not visible in TinyMCE:
	1. Go to the 'Site administration', click on 'Plugins > Text Editors > Editor HTML TinyMCE > General Configurations'.
	2. Insert button 'iassign' in 'customtoolbar'.
